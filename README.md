## 📌 使用建议

1. **PDF阅读**：推荐使用支持横屏、双页显示的阅读器（如 Adobe Acrobat、福昕、iOS 图书等）。
2. **音频播放**：可用于日常课诵、背景听闻或修法时辅助。
3. **文本文件**：`.txt` / `.doc` 文件可在电脑或手机上直接打开，部分文档为白话翻译或原文。
4. **注音版**：特别适合读诵不熟练或初学者的正音辅助。
